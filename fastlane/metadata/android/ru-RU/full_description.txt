C помощью Smart EggTimer вы можете готовить яйца идеально.

Время готовки учитывает

- желаемую консистенцию
- размер яйца
- начальную температуру
- давление воздуха в вашем географическом регионе

Минимальные разрешения:
- Smart EggTimer только просит разрешение для GPS. Это нужно чтобы получить высоту для калькуляции.
  Никакие данные не отправляются по интернету.
